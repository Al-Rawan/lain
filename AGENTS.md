# AGENTS.md

## Project

Node.js project using `@opencode-ai/sdk`.

## GitHub

- Repo: `github.com/Al-Rawan/lain`
- Token: Set `GITHUB_TOKEN` env var

## MCP Servers

Configured in `opencode.json`:
- `github` - GitHub MCP (`@modelcontextprotocol/server-github`)
- `filesystem` - File system access
- `opencode-mcp` - OpenCode integration
- `opencode-toolbox` - Tool search

npm install
